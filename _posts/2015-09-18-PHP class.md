# 类的使用

比如Person类，写在person.php文件中。

	<?php
	    class Person{                                                                       
	        const name='zhh';   
	        public static $age;
	        public static $sex;
	        public $flag=true;     
	        public function __construct(){      //构造函数
	            if ($this->flag){
	                $this->sex='male';
	                $this->age=25;       
	            } 
	            else{
	                $this->sex='female';
	                $this->age=36;  
	            }
	        }                                                                     
	    }
	?>

在另一个文件tryperson.php中调用person.php并输出成员变量。

	<html>
	<body>
	<?php
	include 'person.php';
	$Person1= new Person();
	echo $Person1::name."<br>";
	echo $Person1->age."<br>";
	echo $Person1->sex."<br>";
	?>
	</body>
	</html>

两个文件放在XAMPP的D:\xampp\htdocs\learnPHP目录下，在浏览器中访问tryperson.php，输出结果如下。

	zhh
	25
	male
