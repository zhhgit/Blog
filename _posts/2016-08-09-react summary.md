# 一、使用React的方式

1.通过react-tools将转化jsx

	npm install -g react-tools
	jsx --watch script/ js/
	
2.通过create-react-app初始化一个结合了Webpack和ES6的React工程。

    npm install -g create-react-app
    create-react-app my-app
    cd my-app
    npm start
